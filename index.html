CASHFLOW

Ever need a place to keep track of your finances? Welcome to CashFlow! Each user can log in and access a personalized finance tracker tailored just for them. The app includes a pie chart that summarizes purchases, along with a built-in cryptocurrency converter that provides real-time exchange rates for anyone considering investing.

A full-stack web application built with TypeScript, using React with Vite for the frontend and Express for the backend, styled with CSS. Made for Trends in Web Development course
